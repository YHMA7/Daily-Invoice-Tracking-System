# Daily Installments Invoice Tracking System (DIITS)

A specialized financial management solution developed for the **CPIT-251: Software Engineering I** course at **King Abdulaziz University**. This project addresses the "Liquidity Shock" problem faced by business owners by providing automated daily visibility into combined financial obligations.

---

## 🎯 The Problem & Solution
* **The Challenge**: Business owners managing multiple credit invoices often face sudden financial burdens due to a lack of daily visibility into their total debt.
* **The Solution**: An automated system that breaks down credit invoices into daily payable amounts, redistributing balances instantly after partial payments to ensure continuous cash flow transparency.

---

## 🚀 Key Features
* **Automated Daily Distribution**: Automatically divides invoice totals across the credit period.
* **Dynamic Recalculation**: Instantly redistributes remaining balances after partial payments are recorded.
* **Consolidated Dashboard**: Provides a unified view of total daily commitments across all suppliers.
* **Smart Notifications**: Alerts users about overdue installments and upcoming obligations.
* **Invoice Lifecycle Management**: Tracks invoices from issuance to automatic closure upon full payment.

---

## 🏗️ System Architecture & Design
The system was engineered using a **Layered Architecture** and **Agile Methodology**:
* **Presentation Layer**: Built with **React** and **TypeScript** for a responsive, type-safe user interface.
* **Business Logic Layer**: Handles complex calculations for installment distribution and balance adjustments.
* **Modeling**: Comprehensive documentation including **Use Case Diagrams**, **Sequence Diagrams**, and **Class Diagrams**.

---

## 🛠️ Tech Stack
* **Frontend**: React.js, TypeScript, Tailwind CSS.
* **UI Components**: Radix UI & shadcn/ui.
* **Methodology**: Agile (Iterative development focusing on core high-value features).
